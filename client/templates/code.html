Getting Started

  1. First make sure to include Google Feeds API script preferably in the <head/> section

                            
<script
type="text/javascript"
src="https://www.google.com/jsapi?autoload=%7B%22modules%22%3A%5B%7B%22name%22%3A%22feeds%22%2C%22version%22%3A%221.0%22%2C%22nocss%22%3Atrue%7D%5D%7D"></script>

  2. Now include Angular Feeds files after including AngularJS scripts

    <head>
...
<script src="http://siddii.github.io/angular-feeds/app/angular-feeds/angular-feeds.js"></script>
</head>

  3. Make sure to inject feeds module into your app

    angular.module('angular-feeds-demo', ['feeds',...])
  4. Use <feeds/> directive as you wish...
