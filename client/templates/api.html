Feed Directive

At the very basic, you just need <feed/> directive for displaying the feed content. The only required attribute is the url.
Please note that url should be publicly accessible.

Most of the heavy-weight lifting is done through Google Feeds and all the complexities are completely abstracted to the end user.

Attributes

Name Required Default value
url true Note that it should be a publicly accessible url
summary
 false false
By default, the template displays the full feed content. If this is set to `true`, it will display only the short summary.
count
 false None. Specify the no. of items to display
template-url false Specify the template url for displaying feed content
post-render false Javascript function that will be called after rendering the feed.
